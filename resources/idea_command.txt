CTRL + D - копирование выделенной строки
CTRL + W - выделение слова в строке
CTRL + R - массовое переименование внутри файла
CTRL + SHIFT + R - массовое переименование внутри проекта
CTRL + SHIFT + стрелка вврех или вниз, выделенный блок кода переносит
CTRL + SHIFT + J - объединение строк



SHIFT + ENTER - перенос каретки на новую строчку

SHIFT + ALT + INSERT - Generate Syntaxis


CTRL + ALT + M - создание метода из выделенного блока кода
CTRL + F6 - Изменение сигнатуры метода
CTRL + ALT + T - surround with
CTRL + ALT + U - Diagram
CTRL + ALT + L - форматирование тела файла(например для xml или класса)
CTRL + ALT + V - формирует выражение лямбды в нужную коллекцию + формирует тип переменной для выделенного кода
